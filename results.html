System Usability Scale (SUS) Results

The System Usability Scale (SUS) Results measure the overall usability of the CareSync prototype based on user responses to a standardized 10-item questionnaire. It evaluates key aspects such as ease of use, complexity, confidence, and learnability of the system. The average SUS score provides an overall indication of user satisfaction and perceived usability, helping identify whether the system meets user expectations and where improvements are needed.

DISCUSSION OF FINDINGS

Key Findings

Analysis of both quantitative metrics and qualitative feedback revealed consistent patterns in user behavior and recurring pain points.

Positive Findings

High Task Completion Rate

90% of participants successfully completed the assigned tasks, showing that the system is generally functional and usable.

Good Learnability

SUS results show users found the system easy to learn and understand after initial interaction, especially in completing core tasks.

Helpful Core Feature

Users appreciated the clinic locator and navigation features, especially for showing nearby facilities and basic information.

Positive User Feedback

Participants described the system as practical, simple, and convenient for accessing healthcare-related information.

Areas for Improvement

Slow System Response

Some users experienced lag or delayed feedback when interacting with the system, affecting smoothness of use.

Lack of Users Guide

Users were confused at first due to the absence of a guide or tutorial for navigating the system.

Language Accessibility

Limited language options may affect users who are not comfortable using English-based interfaces.

Interface Clarity

A few users mentioned that some parts of the layout feel slightly confusing or not well structured.

RECOMMENDATIONS

Design Recommendations

Based on the usability results, SUS score, and key findings, the following recommendations aim to improve performance, clarity, and overall user experience while strengthening already positive aspects of the CareSync prototype.

1

Improve System Performance and Reduce Lag

Since some users experienced slow response time, optimize system loading and interactions by improving data handling, reducing unnecessary processes, and adding loading indicators to improve perceived speed.

HIGH PRIORITY
2

Add User Guide

To address user confusion during first use, include a short tutorial or guided walkthrough to help users understand core features such as clinic search and navigation.

HIGH PRIORITY
3

Improve Interface Clarity and Layout Structure

Refine visual hierarchy by improving spacing, grouping related elements, and simplifying layouts to reduce confusion and improve first-time user navigation.

MEDIUM PRIORITY
4

Introduce Multilingual Support for Accessibility

To address language accessibility concerns, add a language toggle (English/Filipino) to make the system more inclusive for non-English users in future iterations.

MEDIUM PRIORITY
5

Conduct Follow-Up Usability Testing

After implementing improvements, conduct another usability test to measure whether task success, satisfaction, and SUS score improve beyond the current marginal rating.

LOW PRIORITY
